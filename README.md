# PictDraw

Десктоп-приложение на Go для создания раскрасок по номерам из SVG-эскизов.

## Что делает
- Загружает входной SVG.
- Находит уникальные цвета заливки фигур.
- Градиенты сводит к одному цвету (берётся цвет первого `stop`).
- Назначает номер каждому цвету.
- Генерирует выходной SVG формата A4 (210×297 мм):
  - фигуры в ч/б (белая заливка, чёрная обводка),
  - номера внутри областей,
  - легенда цветов внизу.

## Запуск
```bash
go mod tidy
go run ./cmd/pictdraw
```

## Ограничения текущей версии
- Номера в сложных `path` ставятся по приближённому центру (среднее по координатам из `d`).
- Генерируется A4 SVG (не PNG/PDF), чтобы сохранить векторность.
